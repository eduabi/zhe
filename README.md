### 简介

zhe是一款基于函数式编程方案的前端JS框架，给你一种原生API急速开发体验，让JS开发更简单纯粹。不在有那么多复杂的概念，大道至简！

----

### 价值目标


>* 学习成本低，可快速上手；
>* 具备开发大型前端架构应用的能力；
>* 具备较高的性能
